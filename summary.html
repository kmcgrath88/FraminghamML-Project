Initial Data


Data was obtained as a .csv file with 11,627 rows of data from 4,434 subjects. Each row contains risk factor and event data for a single subject from a given examination period. Each subject has between 1 - 3 examination periods (roughly 2 years between each examination period for a given subject). For these analyses we focused on period #1 (baseline) as it had the most data (4,434 subjects vs. 3,263 subjects in period #3). For each outcome which we wanted to generate a classifier for we filtered out subjects with the prevalent disease condition (e.g. for the stroke classifier each subject who had already had a stroke prior to their period 1 examination (PREVSTRK = 1) were filtered out of the data set). Filtered out data rows with missing values prior to running classifiers.
